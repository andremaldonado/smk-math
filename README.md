# SMK-Math

SMK-Math is a simple webpage for my son to study some math problems.

## Installation

* Clone the repo
* Install gulp (it it is not installed) `npm install -g Gulp`
* Install dependencies `npm install`

## Usage

You can simple open src/index.html
